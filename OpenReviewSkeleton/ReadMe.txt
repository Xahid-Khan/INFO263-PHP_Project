Run locally:

1. Download ampps: https://ampps.com/ (take note of installation path)
2. Extract project archive in ampps/www
3. Run amps and click on the Globe button

If this doesn't work:

Download and install PHPStorm: https://www.jetbrains.com/phpstorm/

4. Open the project in PHPStorm by clicking open and pointing at project folder
5. Open index.php and in the top right corner click on the icon of your preferred browser
6. If that doesn't work, click on configure PHP in the bottom right corner
7. Set the path to php.exe
